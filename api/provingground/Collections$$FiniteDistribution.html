FiniteDistribution

Related Docs: object FiniteDistribution | package Collections

case class FiniteDistribution[T](pmf: Seq[Weighted[T]], epsilon: Double = 0.0) extends ProbabilityDistribution[T] with LabelledVector[T] with Product with Serializable

Finite distributions, often supposed to be probability distributions, but may also be tangents to this or intermediates.

pmf

probability mass function, may have same object split.

epsilon

cutoff below which some methods ignore objects. should be very small to allow for split objects.
