AgdaExpressions 

object AgdaExpressions

Parsing an Agda-like language in two steps. First there is a combinator parser that recognizes agda tokens. At the next level, we note names and try to parse to terms.

There is also an object to recognize agda tokens.

  1.  class AgdaParse extends JavaTokenParsers

    Parses an agda expression.

    Parses an agda expression. following scala, we first look for a valid expression without including line breaks, then include line breaks.

    tokens cannot contain whitespaces or parenthesis, and also cannot be reserved words/symbols.

    parses expressions, defns, data statements.

    have helpers for space, whitespace, end of paragraph, to mapsto etc.
