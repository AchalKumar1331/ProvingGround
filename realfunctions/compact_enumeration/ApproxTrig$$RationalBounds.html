class RationalBounds extends ApproxTrig with ElementaryFunctions[Approx]

Bounds for elementary functions on a cube; bounds for sin, cos etc are a function on an interval; bounds for coordinates only depend on the cube. Combining the two should be done with care (as is done in rationalBound)
