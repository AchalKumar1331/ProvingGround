    The main recursive translator:

    split

    matches and optionally splits to a type XI derived from I, possibly starting with a sub-type of I.

    join

    optionally combined from a type XO built from O, possibly starting with a sub-type of O. inference of I and O is difficult in this case, so should generally not be used directly. Instead when combining translations I and J are determined by the base, and the rest of the type parameters by the split and join.
